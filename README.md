## Running The Project Locally

The setup for this project is very straightforward <br>
<br>

#### Clone The Project

```sh
git clone https://github.com/DavidHDev/react-bits.git .
```

#### Install Dependencies

```sh
npm install
```

#### Start The Development Server

```sh
npm run dev
```
